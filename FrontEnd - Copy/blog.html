Introduction to Java

January 1, 2022 by Abraham

Java is a popular general-purpose programming language. It runs on servers, computers, ATM's, printers, and many other devices. Netflix and Twitter are two examples of large companies that use Java extensively.

There are many programming languages in the world. What makes Java so special? Java programs rely on another program, the Java Virtual Machine (JVM) to run.

The JVM takes care of platform-specific details which makes writing programs in Java much easier than some other languages.

he developers of Java follow a principle known as "Write Once, Run Anywhere." This means they can write a Java app once on their computer and share it with anyone who is running a Java Virtual Machine on their computer.

As you might imagine, managing one body of code is easier than monitoring two or more.

The Eight Primitive data types

Primitives are data types supported directly in the Java language.

Primitive data type are byte, short, int, long, float, double, char, boolean.

Methods

A Java method is a collection of statements that are grouped together to perform an operation. Some other languages refer to this as a function or a procedure. Methods model the behavior in a class.

Methods also begin with a declaration referred to as the method signature, followed by an opening curly brace { then one or more statements, finally ending in a closing curly brace }. The statements within the method execute in a top down fashion, so their ordering must follow specific rules that the compiler will check for.

public static void main(String[] args) {
        }

The main() is the starting point for JVM to start execution of a Java program. Without the main() method, JVM will not execute the program. Above is the syntax of the main() method.

Source files

A source file is a text file with the .java extension that you save your code in.

A single package can have many source files or even just one. Source files in Java must follow certain rules. A source file typically has only one class in it and has the same name as the class. However, you can have source files that have more than one class. Typically this is only done if you have specialized supporting classes that support the main class.

Classes

August 23, 2023 by Emely

A class is a source code template that describes the state and behavior of an object. An object is what the Java Virtual Machine creates in memory from the class template.

Classes start with a class declaration, followed by an opening curly brace : { . This is followed by the body of the class which contains methods and variable declarations. The variables in a class model state and the methods model behavior. Finally, the class body is terminated with a closing curly brace: } .

Methods in a class can be called in any order or not at all. There is no need for the methods to be in a specific order as far as the compiler is concerned, but it is considered good practice to follow some basic rules which you will learn shortly. Your fellow programmers will be very grateful.

Client-Side Tech: JavaScript

July 29, 2022 by Brenton

JavaScript is often confused for Java and vice-versa. This confusing name is not entirely an accident. When JavaScript was first being developed at Netscape, it went through the names Mocha and LiveScript. The company wanted it to have wider appeal. To do this, they licensed the name Java from Sun Microsystems and renamed LiveScript to JavaScript. This allowed Netscape to piggyback off the success of the rapidly expanding Java language. So, while the names are similar, there is very little in common between the two.

JavaScript has since undergone a lot of changes as its importance to the Internet has expanded. JavaScript is now a standard under the ECMA organization and is referred to as ECMAScript. You've probably seen JavaScript referred to as ES5, ES6, or ES2016. That is short for ECMAScript 5/6/2016, which refers to various versions of the standard.

Data Types

You can complete many tasks with JavaScript. When carrying out these tasks, you work with different types of data. There are three main types of data: String, Number, and Boolean.

  • A String is a group of words, like a sentence. When writing a String in JavaScript, you need to wrap the group of words you are using within quotes.
  • A Number is just that, a number. Unlike a String, a Number does not need to be wrapped in anything. If you write out a number, like 24, it will be read as a number automatically.
  • A Boolean is a logical entity and can only have two values: true and false. A Boolean is like a Number in that it does not need to be wrapped in quotes. If you set something to true without quotes, the value will be interpreted as Boolean.
